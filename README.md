# Система управления ПТФ и шторкой дальнего света v22.1

[![Arduino](https://img.shields.io/badge/Arduino-ATmega328P-blue)](https://www.arduino.cc/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Автомобильный контроллер для плавного управления противотуманными фарами (ПТФ) и шторкой дальнего света с защитой АКБ, вежливой подсветкой и энергосбережением.

## 🚗 Возможности

- ⚡ **Плавное включение/выключение** ПТФ через ШИМ (1250 Гц)
- ⏱ **Задержка включения ДХО** (10 секунд после запуска)
- 🕒 **Два режима задержки выключения:** 10 сек / 30 сек (вежливая подсветка)
- ☀ **Автоматическая яркость:** 70% (ДХО) / 100% (ПТФ/габариты)
- 🎭 **Управление шторкой** дальнего света с антидребезгом
- 🔋 **Защита АКБ** от глубокого разряда
- 💤 **Режим глубокого сна** (Power-down)
- 🛡 **Watchdog Timer** для защиты от зависаний
- 🚗 **Питание от постоянного плюса АКБ**

## 🔌 Подключение

| Пин Arduino | Назначение         |
|-------------|--------------------|
| D10 (ШИМ)   | Яркость ПТФ        |
| D6          | Шторка дальнего    |
| D2          | Габариты           |
| D4          | Дальний свет       |
| D7          | Кнопка ПТФ         |
| D8          | Сигнал генератора  |
| A0          | Напряжение АКБ     |

## 🛠 Аппаратная защита

- **SS34** — диод Шоттки (от переполюсовки)
- **SMAJ18A** — TVS-диод (от бросков напряжения)
